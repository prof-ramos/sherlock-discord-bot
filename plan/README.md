# Planejamentos (Plans)

Este diretório armazena planejamentos em andamento/não finalizados.

## Regras

1.  **Planejamentos Não Finalizados**: Use este diretório para esboçar, estruturar e detalhar
    tarefas complexas.
2.  **Limpeza**: Assim que um planejamento for concluído e implementado, **o arquivo correspondente
    deve ser apagado**.
3.  **Organização**: Mantenha os nomes dos arquivos descritivos (ex: `refatoracao-auth.md`,
    `nova-feature-rag.md`).
